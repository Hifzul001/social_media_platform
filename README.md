# Django Social Media Platform

A full-stack social media web application built with Django.

## Features
- User Authentication
- User Profiles
- Create, Edit & Delete Posts
- Like & Comment System
- Follow/Unfollow Users
- News Feed
- Responsive UI

## Tech Stack
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## Installation

```bash
git clone <repository-url>
cd <repository-name>
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
